# Hands-On Large Language Models - Learning Path

This repository serves as a hands-on learning path for exploring Large Language Models (LLMs), based on the book *Hands-On Large Language Models* by Jay Alammar & Maarten Grootendorst.

## 📚 Syllabus & Progress Tracker

### Part I: Understanding Language Models
This section covers the inner workings of language models, starting from basic representations up to the Transformer architecture.

- [ ] **Chapter 1: An Introduction to Large Language Models**
  - Topics: History of Language AI, Bag-of-Words, Dense Vector Embeddings, Encoders vs. Decoders.
- [ ] **Chapter 2: Tokens and Embeddings**
  - Topics: Tokenization, Subwords, Contextualized Word Embeddings, Word2Vec.
- [ ] **Chapter 3: Looking Inside Large Language Models**
  - Topics: Transformer blocks, Attention Mechanisms, Decoding Strategies.

### Part II: Using Pretrained Language Models
This section focuses on practical applications using existing pretrained models without needing to fine-tune them.

- [ ] **Chapter 4: Text Classification**
  - Topics: Supervised & Zero-shot Classification, Representation vs. Generative models.
- [ ] **Chapter 5: Text Clustering and Topic Modeling**
  - Topics: Dimensionality Reduction, HDBSCAN, BERTopic.
- [ ] **Chapter 6: Prompt Engineering**
  - Topics: Instruction prompting, Chain-of-thought, Constrained sampling.
- [ ] **Chapter 7: Advanced Text Generation Techniques and Tools**
  - Topics: LangChain, Conversational Memory, ReAct Agents.
- [ ] **Chapter 8: Semantic Search and Retrieval-Augmented Generation**
  - Topics: Dense Retrieval, Reranking, RAG, Evaluation.
- [ ] **Chapter 9: Multimodal Large Language Models**
  - Topics: Vision Transformers (ViT), CLIP, BLIP-2, Image Captioning.

### Part III: Training and Fine-Tuning Language Models
This section dives into advanced concepts covering training and fine-tuning.

- [ ] **Chapter 10: Creating Text Embedding Models**
  - Topics: Contrastive Learning, SBERT, TSDAE for Domain Adaptation.
- [ ] **Chapter 11: Fine-Tuning Representation Models for Classification**
  - Topics: Few-Shot Classification (SetFit), Masked Language Modeling, NER.
- [ ] **Chapter 12: Fine-Tuning Generation Models**
  - Topics: Supervised Fine-Tuning (SFT), PEFT/LoRA, QLoRA, Preference Tuning (DPO).

## 🚀 Getting Started
Check out the folders `01_understanding_llms`, `02_using_pretrained_models`, and `03_training_and_finetuning` to find Jupyter notebooks corresponding to each chapter!
